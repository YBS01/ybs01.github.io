vMix Setup Instructions for Telethon Tracker 2026

Adding the Graphic to Your Preset:

    Click "Open" in the top-left corner of vMix
    Locate the SARM 2026 Telethon Graphic .vmix file
    When the dialog appears, click “Append” (bottom-right)

Displaying the Graphic:

    Activate the tracker using one of vMix's overlay channels

Controlling the Tracker:

    Right-click the "User Interface - Tracker" input
    Enable "Mouse and Keyboard" interaction
    This allows you to set the goal amount and update the current total

Optional - Automatic Updates:

    You can link a Google Sheet to automatically tally pledges and update the tracker
